# Algonive Sales Analysis

This project analyzes Superstore sales data.

## Folder Structure

- `notebook/`: Contains the Jupyter notebook for data analysis (`sales_analysis.ipynb`).
- `dataset/`: Contains the raw dataset (`Superstore Sales.xlsx`).
- `output/`: Contains the processed output file (`superstore_final.csv`).

## Usage

1. Open `notebook/sales_analysis.ipynb` to view the analysis.
2. The raw data is available in `dataset/Superstore Sales.xlsx`.
3. The final output is saved in `output/superstore_final.csv`.
